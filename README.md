\# EduScoreX



AI-Powered Academic Analytics Platform



\## Features



\- Student Management

\- Attendance Tracking

\- AI Study Recommendations

\- Leaderboard Analytics

\- Rank Dashboard

\- Report Card Generation

\- Role-Based Authentication

\- Teacher \& Student Portals



\## Tech Stack



Frontend:

\- React

\- Tailwind CSS

\- Framer Motion



Backend:

\- Node.js

\- Express.js

\- MongoDB



\## Installation



\### Frontend



npm install

npm run dev



\### Backend



npm install

npm run server
